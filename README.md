# \# AutoMouseClicker



This is a tool base on c **only** suit for windows

这是一个基于c语言的项目**仅**支持Windows



download the c files **"AutoMouseClicker2.c"** and compile it on gcc



you need to input

**gcc AutoMouseClicker.c -o xxxxxxx.exe -mwindows -luser32 -lkernel32**

to compile 32-bit programme on windows



**x86\_64-w64-mingw32-gcc AutoMouseClicker2.c -o xxxxxxx.exe -mwindows -luser32 -lkernel32**

to compile 64-bit programme on windows
